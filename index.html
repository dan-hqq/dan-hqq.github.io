About

Halo! Perkenalkan nama saya Octa Ramadhani Rinantoro, biasa dipanggil Dani. Saya lahir pada tanggal 23 Oktober 2004 di Kota Kediri. Saya adalah anak kedua dari dua bersaudara. Saya memiliki hobi membaca komik dan bermain futsal. Komik saya baca biasanya adalah komik Jepang, China, dan Korea. Saya suka bermain bola sudah dari kecil. Ayah saya suka mengajak saya bermain bola saat akhir pekan. Pada waktu SMP dapat hampir setiap hari bermain bola bola dengan teman saya di sekolah.
Sekarang saya berkuliah di jurusan D3 Teknik Informatika PENS. Alasan saya memilih Informatika adalah karena saya memimliki minat terhadap dunia komputer/informatika sejak kecil. Sejak kecil saya sudah memiliki cita-cita menjadi programmer dan sejak kecil saya juga menyukai matematika. Saya rasa kesukaan saya terhadap matematika membantu saya saat berkuliah di Informatika karena dapat melatih logika. Dan Alasan saya memilih PENS adalah karena pada awalnya saya tidak diterima di SNMPTN dan SBMPTN.
